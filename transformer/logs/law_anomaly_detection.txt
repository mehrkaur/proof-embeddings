Experiment # 1:
---------------
Description: Using the masked language model (MLM) that is trained on a normal dataset that
consists of laws of i) logical equivalence, ii) commutativity, iii) demorgan, iv) tautology, and
v) associativity, we computed the masked language modeling loss (i.e. CLOZE task) on test set
that contains the held-out sample portion (i.e. test set) aforementioned laws, but additionally also
vi) fallacy. Here, the novel/outlier/anomaly data instances are fallacy, and the rest of the laws
fall under normal data instances. Similar to what is often done in an unsupervised anomaly detection
setting where they train an autoencoder (AE) on normal data, and try to detect anomalies via
reconstruction loss -- the anomalies would yield higher losses. The specific loss we are computing
is aggregated sequentially; we mask each token one step at a time, going left-to-right, and try to
predict the token and make note of the loss at each timestep via the standard cross-entropy loss.
Just like in the literature, we will try to find a threshold T for loss, which we hypothesize that
normal data yields t <= T and anomalies yield t > T. To find such a T rather quickly and easily,
we can just look at the statistics of each law over thousands of examples. This would of course
work accurately, roughly speaking, when losses for the anomaly class (+) is higher on average than
normal classes (-). To make sure that the unseen token 'F' (False) during training (because the
train set only consisted of tautology-generated data) does not bias predictions of the model
(worst-case scenario would be if the model was a Markov Chain), we replace it by 'T'. We also
evaluate with replacing it by '[UNK]' and achieve similar results.
